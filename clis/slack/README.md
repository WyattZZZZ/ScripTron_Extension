# Slack CLI

Official ScripTron wrapper for the **Slack CLI** (`slack`).

## 📌 Prerequisites
```bash
./install.sh
```

## 🔐 Authentication (OAuth)
```bash
./run.sh --action login
```
Opens a browser window to authorize the CLI with your Slack workspace.

## 💬 Usage
```bash
./run.sh --action run --input "app list"
./run.sh --action run --input "deploy"
```
