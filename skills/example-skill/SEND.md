# Send Sub-Skill Workflow

## 适用场景 (Use Cases)
Sending messages or data to a specified destination.

## 输入要求 (Input Requirements)
Destination address and message payload.

## 工作流步骤 (Workflow Steps)
1. Validate the destination address.
2. Format the message payload.
3. Call `example-cli send` command.
4. Verify transmission status.

## 输出格式 (Output Format)
Transmission receipt or error log.
